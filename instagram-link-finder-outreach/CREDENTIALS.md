# Credentials required

- **Google Sheets OAuth2 credential**: used by "Get row(s) in sheet" and "Update row
  in sheet" to read a lead list and write back research results.
- **Perplexity API credential**: used by "Message a model" to send each lead to a
  Perplexity model for Instagram lookup and website quality evaluation.

The Google Sheet needs columns for at least: Name, Address, Website, Instagram,
Confidence, Reason, Website Quality, and Website Reason.
