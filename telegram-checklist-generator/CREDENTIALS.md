# Credentials

- **Telegram Bot API**: used by the "Telegram Trigger" node to receive incoming messages, and by both "Send a text message" nodes to send the checklist and the paced individual task messages back.
- **Google Gemini API** (Google PaLM/Gemini credential type in n8n): used by the "Google Gemini Chat Model" node as the language model behind the AI Agent that extracts tasks from the incoming message.
